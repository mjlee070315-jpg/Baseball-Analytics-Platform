# ⚾ MLB Analytics Platform

A full-stack style baseball analytics dashboard built with React and MLB Stats API.

Search MLB players, view real-time statistics, analyze performance trends, and explore team/game information through an interactive interface.

## 📌 Features

### 🔎 Player Search
- Search MLB players using MLB Stats API
- View player profile information
- Display real-time player data

### 📊 Player Analytics Dashboard
- Individual player analysis page
- Hitting statistics:
  - AVG
  - OPS
  - HR
  - RBI

- Pitching statistics:
  - ERA
  - WHIP
  - Wins
  - Strikeouts

### 📈 Data Visualization
- Season statistics bar charts
- Game performance trend charts
- Interactive Recharts visualization

### ⚾ Game Center
- Today's MLB games
- Live game information
- Game detail page

### 🏆 Standings
- Real-time MLB standings
- Division rankings
- Team details

### 🔥 Player Comparison
- Compare two MLB players
- Compare statistical performance
